# Mojang
Library for easy usage of Mojang heads &amp; profiles

# Maven
```xml
<repositories>
    <repository>
        <id>scarsz</id>
        <url>https://nexus.scarsz.me/content/groups/public/</url>
    </repository>
</repositories>

<dependency>
  <groupId>github.scarsz</groupId>
  <artifactId>mojang</artifactId>
  <version>1.0</version>
</dependency>
```
